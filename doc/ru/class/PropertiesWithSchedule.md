PropertiesWithSchedule
===============
Источник: [AIXM](https://extranet.eurocontrol.int/http://webprisme.cfmu.eurocontrol.int/aixmwiki_public/bin/view/AIXM/PropertiesWithSchedule)

Тип: Object
